Definição de css

O css é utilizado para estilarizar as nossas páginas em html,controlando a forma que uma página é estruturada para obter uma melhor aparência.

Há três formas de aplicar o css sendo elas:

  • Inline:Estilos que são aplicados diretamente em um elemento html
  • Incorporado: modo utiliza o comando style dentro de uma página html para aplicar
    as estilizações html o seu comando é style tipo de elemento que deseja estilarizar /style
  • Folha de estilo externa: você cria uma classe com a extensão .css, fazendo com que você possa estilizar uma pagina inteira ou mais sem ter que poluir o seu código html.

A sintaxe do css para que possamos estilizar a nossa página é selector{propriedade:valor;}

Por exemplo queremos mudar a cor de um paragrafo para vermelho iremos escrever p{color:red;}.

Ou se quisermos mudar o alinhamento de um h2 escreveremos h2{align:center;}

Mas e se quisermos mudar todos os elementos de um documento, como que faz??
E eu te respondo meu leitor usaremos o Seletor Universal onde sua sintaxe é *
que representa todos os elementos do documento.

Mas e se tivermos o exemplo que temos toda a página com os paragrafos da cor azul e queremos
que um único paragrafo tenha a cor azul, como que fazemos?
Ai usaremos o seletor de classe ou seletor de id.

Seletor de classe: você cria uma "variavel" dentro do lugar onde você quer modificar
por exemplo p, h2,h3,ul,etc...
.nome-da-variavel{color:red;} você irá aplicar cor vermelha a todos os elementos que possuem o atributo class = "nome-da-variavel"
p.nome-da-variavel{color:red;} você irá aplicar a cor vermelha a todos os p que possuem o atributo class="nome-da-variavel"

Seletor por id: Funciona da mesma maneira que o seletor de classe porém usaremos id e a sintaxe
ao inves de ser .(ponto) usaremos #(hashtag), por exemplo:
#nome-do-id{color:cian;} aplica a cor ciano a todos os elementos que possuem o id = nome-do-id
h2#nome-do-id{color:cian;} aplica a cor ciano a todos os h2 que possuem id = "nome-do-id"

Algumas observações importantes para criarmos ids ou classes:

  • Não usar espaços em branco
  • Não utilizar caracteres especiais
  • Ele é case-sensitive
