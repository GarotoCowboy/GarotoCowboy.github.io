Projeto em Angular

Meu projeto utilizando angular será um virtual tabletop.

O que é um virtual table top?

Um tabletop é um estilo de plataforma muito utilizada para jogar rpg de mesa virtual, será possivel fazer rolagens de dados, adicionar ficha dos jogadores, terá um tabuleiro onde poderão adicionar uma imagem representando seus personagens e move-los pelo tabuleiro. farei inspirado no site https://roll20.net/welcome se for possivel utilizar banco de dados, terá também um sistema de login para que o jogador que for mestrar possa adicionar seus jogos e adicionar os seus jogadores a ele. O meu projeto em sala também será esse mesmo.
